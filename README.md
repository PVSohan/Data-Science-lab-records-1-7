# Data Science Lab Records 1-8

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange)](https://jupyter.org/)

Repository containing lab records for Fundamentals of Data Science course (Lab 1-8). Each lab contains practical implementations and programs related to various data science concepts.

## Features

- 8 comprehensive lab notebooks
- Hands-on implementations of data science concepts
- Clean and documented code
- Practical examples with datasets
- Progressively complex implementations

## Installation

1. Clone the repository:
```bash
git clone https://github.com/PVSohan/Data-Science-lab-records-1-7.git
```

2. Install requirements:
```bash
pip install -r requirements.txt
```

## Usage
Run Jupyter Notebook:
```bash
jupyter notebook
```
Navigate to the desired lab notebook:
- `FDS LAB-1 AV.EN.U4CSE22036.ipynb`
- `FDS LAB-2 AV.EN.U4CSE22036.ipynb`
- ...through LAB-8
